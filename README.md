# Sobre

Este é um repositório feito para documentar minha evolução na área de programação. Sou estudante de programação, atualmente (no tempo que escrevi este texto) tenho 24 anos e estou tentando ingressar na área, tenho alguns conhecimentos prévios sobre programação, mas devido ao excesso de informação acabei me perdendo. 

Nesta minha jornada, duas coisas me auxiliaram para entender o processo e conseguir vencer o pensamento "Será que programação é para mim?", a primeira ela sendo o repositório: < https://github.com/jwasham/coding-interview-university >, onde ele traz um roadmap, para a linguagem e conhecimentos necessários para concorrer uma vaga em grandes organizações. A segunda foi ler o livro `Programador Pragmático - Andrew Hunt; David Thomas`, esse livro sendo a grande virada de chave, sobre entender postura profissional, mentalidade em ambientes de desenvolvimentos, e gestão de projetos.

Este repositório será separado por areas de conhecimento, sendo linguagens de programação, bancos de dados e acréscimos.
